Task 3: CSS Units, Box Models, and Fonts
A simple web component featuring an image and a button, styled using CSS Viewport Units and Box Model properties without using Flexbox or absolute positioning.

 Features
Centered Layout: Uses margin and viewport units for precise screen positioning.

Viewport Units: Responsive sizing using vw (viewport width) and vh (viewport height).

Custom Styling: A button with a linear gradient background and custom font settings.

No Scroll: Implemented with border-box sizing to ensure no layout overflow.

 Requirements Met
[x] Equal spacing of 25% from left and right.

[x] Spacing of 10% from top and bottom.

[x] No Flexbox used.

[x] No Position: relative/absolute used.

[x] No Overflow: hidden used.
